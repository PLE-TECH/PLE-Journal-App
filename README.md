# 📝 Personal Journal App

A simple, interactive personal journal built using **vanilla JavaScript**, **HTML**, and **CSS**. This project was developed as part of the **Week 5 JavaScript Fundamentals Assignment**.

---

## 🎯 Project Goal

To demonstrate foundational skills in:
- Structuring web pages using HTML
- Styling with CSS
- Using JavaScript to dynamically interact with the DOM

---

## 📦 Features

- ✅ Static HTML/CSS layout with a header, dummy journal form, and list display
- ✅ Hardcoded journal entries in JavaScript
- ✅ Dynamic rendering of entries to the DOM
- ✅ "Toggle Details" button to show/hide each entry's content

---

## 📚 Week 5 Deliverables

### 1. HTML/CSS Layout [COMPULSORY]
- Header with project title
- Entry form layout (currently disabled)
- Journal entry list area

### 2. JavaScript Functionality
- ✅ Hardcoded array of journal entries (title, date, content) [COMPULSORY]
- ✅ Loop through entries and render them in the DOM [OPTIONAL]
- ✅ "Toggle Details" button to show/hide entry content [OPTIONAL]
